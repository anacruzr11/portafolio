Aplicación CRUD

El proyecto representa una aplicación CRUD para un concurso de un kit de maquillaje totalmente gratis. Los usuarios ingresan su nombre completo, correo electrónico y el número de teléfono. Los datos aparecerán en una tabla debajo del formulario en donde el usuario tendrá la posibilidad de modificar o eliminarlos.

La página es completamente responsive, es decir, se adapta a diferentes formatos de pantallas, desde computadores grandes hasta tablets y celulares pequeños.

El proyecto fue desarrollado con el uso de HTML, CSS y Bootstrap v5.3.0
